# ACMEMarketing
Este proyecto está diseñado para resolver el problema de gestión de visitas comerciales de *ACME*, una empresa de venta de productos de oficina. Los comerciales registran las visitas a clientes en una hoja de Excel compartida, lo que puede generar confusión y dificultades para mantener el control y seguimiento de la información.

La solución propuesta está desarrollada en [.NET Core](https://dotnet.microsoft.com/es-es/download/dotnet/3.1) y ofrece una plataforma que permite a los usuarios visualizar, agregar, editar y eliminar registros de visitas comerciales. Los principales elementos gestionados incluyen:

1. Cliente visitado
2. Fecha de visita
3. Comercial responsable
4. Datos adicionales que puedan ser necesarios

La aplicación está desarrollada con un diseño adaptado a las necesidades del negocio, asegurando una interfaz fácil de usar y un flujo de trabajo optimizado para la gestión de visitas comerciales.

Este proyecto fue creado utilizando [Visual Studio Code](https://code.visualstudio.com/) y se centra en ofrecer una solución moderna, eficiente y escalable.
